/**
34 
 * Tries to compute a list of variables that can shadow a variable in the
35 
 * outer scope.
36 
 *
37 
 * For example:
38 
 *
39 
 * <code>
40 
 * var a = function() {
41 
 *   var b = getB();
42 
 *   b();
43 
 *   return function(y) {};
44 
 * };
45 
 * </code>
46 
 *
47 
 * Normally, b would be mapped to variable L0, y would be L1.
48 
 *
49

 *
